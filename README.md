#Taffy_Web_Py3.x 

# 1. 运行环境
- macOS，linux，windows
- python 3.x

# 2. 相关lib安装及配置
```
pip install flask
pip install flask-bootstrap
pip install flask-wtf
pip install nose
pip install nose-html-reporting
pip install pyyaml
```

# 3.运行

```
# python run.py
```

使用浏览器打开http://127.0.0.1:5000
